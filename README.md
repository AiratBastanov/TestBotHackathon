Hackathon_project AI это Telegram Bot с Voice-to-Text и фильтрацией контента, поддержкой PDF/DOCX/TXT анализа, а также системой расширяемых плагинов (погода, курсы валют).
#Использованные технологии
- Технологии
- Python 3.12
- Telegram API (python-telegram-bot v20)
- DeepSeek API
- aiohttp
- PyPDF2, python-docx
- pydub
- dotenv
- модульная архитектура
#Фильтрация сообщений идёт по 12 параметрам:
- Нецензурная лексика
- Ссылки, рекламу, контакты
- Спам
- Взрослый контент
- Наркотики
- Мошенничество
- Хейтспич
- Повторения, капслок и др.
Все нарушения блокируются автоматически
#Анализ файлов (PDF / DOCX / TXT)

После загрузки файла бот:
1. Извлекает текст (PyPDF2 / python-docx / UTF-8)
2. Проверяет на нарушения
3. Предлагает тип анализа:
   - Пересказ  
   -  Ключевые пункты  
   - Подробный анализ  
   - Вопросы и ответы  

#Голосовые сообщения
- Распознаёт речь пользователя
- Поддерживает анализ содержания
- Использует `voice_processor.py`

#Плагины
Бот автоматически подключает плагины (если доступны):

- Погода (city → прогноз)
- Курс валют
- Единая система управления: `plugins/init.py` (Любой файл в папке plugins/ автоматически подключается)
Работает даже если плагины отсутствуют.

# Установка и запуск
#1️Установка зависимостей
pip install -r requirements.txt
# 2️Создайте файл `.env`
В корне проекта создайте файл `.env` и добавьте туда ваши ключи:
``env
TELEGRAM_TOKEN=ваш_telegram_token
DEEPSEEK_API_KEY=ваш_api_ключ_от_DeepSeek
#3 Команды бота
Команда: /start
Описание: Запускает бота.
Команда: /help
Описание: Показывает список доступных функций.
Команда: /stats
Описание: Предоставляет техническую информацию: время ответа, количество токенов и другие данные.
Команда: /reset
Описание: Очищает контекст диалога.
Все остальные команды:
Описание: Любой введенный текст фильтруется и отправляется в DeepSeek.


Для работы с голосовыми сообщениями используется FFmpeg для конвертации аудиоформатов (программа для конвертации аудио/видео файлов. Боту нужна для преобразования голосовых сообщений в текст), установка идёт через Chocolate (распаковка в C:\ffmpeg\, добавление по пути C:\ffmpeg\bin в PATH).
#Используемые API
Внешние сервисы API

Погодные сервисы
OpenWeatherMap API:

Эндпоинт: https://api.openweathermap.org/data/2.5/weather

Назначение: Получение текущей погоды

Особенности: Поддержка города Казань

Финансовые API
Центральный Банк РФ:

Эндпоинт: https://www.cbr-xml-daily.ru/daily_json.js

Назначение: Актуальные курсы валют

Особенности: Официальные курсы USD, EUR

Telegram Bot API
Основной API Telegram

Доступ: Через python-telegram-bot библиотеку

Основные функции:

Прием и отправка сообщений

Обработка файлов и изображений

Управление клавиатурами и кнопками

Webhook/Long Polling соединение

Health checks - проверка работоспособности

Лимиты и ограничения
DeepSeek: Зависит от тарифного плана

Telegram: 30 сообщений/секунду на бота

OpenWeatherMap: 1000 запросов/день (бесплатно)

ЦБ РФ: Без ограничений
